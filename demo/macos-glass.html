核心实现原理

  • backdrop-filter: blur(20-40px) 毛玻璃模糊，数值越大越"磨砂"
  • saturate(1.5+) 提升饱和度，让模糊后的颜色不会变灰暗
  • background: rgba(255,255,255, 0.08~0.15) 半透明白底叠加
  • border: 0.5px solid rgba(255,255,255, 0.2~0.3) 极细白边模拟玻璃边缘高光
  • inset 0 1px 0 rgba(255,255,255, 0.25) 内阴影实现顶部亮线
  • conic-gradient 锥形渐变实现金属光环旋转效果（+ 按钮边框）
  • -webkit-mask-composite: xor 将渐变裁剪为环形边框
