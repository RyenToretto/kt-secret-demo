Glassmorphism vs Liquid Glass

左为传统毛玻璃 (2020)，右为液态玻璃 (2025)。核心区别在于折射层和镜面高光。

Glassmorphism
传统毛玻璃
仅 backdrop-filter: blur() + 半透明背景 + 边框。 视觉上是平面的，没有深度和光学效果。
Liquid Glass
液态玻璃
SVG feDisplacementMap 折射 + 多层高光 + inset box-shadow。 模拟真实曲面玻璃的光线弯曲与边缘反射。
原理

四层架构

Liquid Glass 的 CSS 实现基于四层叠加，每层负责一个光学特性。

Liquid Glass = 折射 + 色调 + 高光 + 内容

1

折射层 (Refraction)

SVG feTurbulence 生成噪声 → feGaussianBlur 平滑 → feDisplacementMap 位移背景像素，模拟光线穿过曲面玻璃时的弯曲。通过 backdrop-filter: url(#filter) 应用。仅 Chrome 完整支持，其他浏览器降级为纯 blur。

2

色调层 (Tint)

background: rgba(255,255,255, 0.05~0.1) 半透明白色叠加，让玻璃有微微的"体积感"，同时自动拾取背景色调（Apple 称为 Adaptive Tinting）。

3

镜面高光层 (Specular)

inset 0 1.5px 1px rgba(255,255,255,0.25) 内阴影模拟顶部亮线。border: 0.5px solid rgba(255,255,255,0.25) 模拟玻璃边缘高光。顶部用 ::before 伪元素绘制集中的水平高光条。

4

内容层 (Content)

实际的文字、图标等 UI 内容，z-index 置于最顶层。Apple 的设计哲学：内容永远是主角，液态玻璃只是"让路"的容器。
